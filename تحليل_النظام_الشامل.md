## 📋 ملخص تنفيذي

**AZRAR** هو نظام إدارة عقارات متطور يعمل على سطح المكتب، تم تطويره باستخدام تقنيات حديثة وآمنة. يوفر النظام حلاً شاملاً ومتكاملاً لإدارة العقارات والعملاء والعقود والمدفوعات مع واجهة مستخدم عربية كاملة.

### النقاط الرئيسية

- **النوع:** تطبيق سطح المكتب (Electron + React)
- **الهدف:** إدارة شاملة للعقارات والمعاملات العقارية
- **اللغة:** دعم كامل للغة العربية (RTL)
- **قاعدة البيانات:** SQLite محلية مع إمكانية المزامنة مع SQL Server
- **الأمان:** نظام صلاحيات متعدد المستويات (RBAC) + تشفير + ترخيص

## 🔐 الأمان والصلاحيات

### نظام RBAC (Role-Based Access Control)

#### الأدوار المتاحة
```typescript
1. SuperAdmin - المدير الأعلى
   - صلاحيات كاملة بدون قيود
   - إدارة المستخدمين
   - حذف البيانات
   - الوصول للإعدادات الحساسة

2. Admin - مدير النظام
   - معظم الصلاحيات
   - إدارة البيانات
   - إضافة/تعديل/حذف
   - عرض التقارير

3. Employee - موظف
   - عرض البيانات
   - إضافة/تعديل محدود
   - تسجيل الدفعات
   - إنشاء العقود

4. Accountant - محاسب
   - عرض مالي كامل
   - إدارة الدفعات
   - التقارير المالية
   - محدود في التعديل
```
