## الإصدار 3.3.0 (Smart Dashboard Upgrade)

### 📊 إضافات لوحة القيادة (Dashboard Widgets)
- **Marquee Tracking Bar**: شريط تتبع ذكي يعرض التنبيهات، العقود المنتهية، والدفعات المستحقة بشكل متحرك.
- **Client Tracking System**: ويدجت جديد لمتابعة آخر التفاعلات مع العملاء (اتصالات، شكاوى، زيارات).
- **Dashboard Calendar**: تقويم مصغر تفاعلي يعرض الأحداث اليومية (دفعات، انتهاء عقود، تذكيرات).
- **Sticky Notes**: نظام ملاحظات سريعة مع تحديد الأولويات.
- **Smart Reminders**: نظام تذكير ذكي للمهام اليومية مع إمكانية التعليم كمنجز.
- **Follow-Up System**: ويدجت لإدارة جدول المتابعات اليومية مع العملاء.

### ⚙️ التحسينات التقنية
- إضافة `DashboardWidgets.tsx` لتجميع كافة الويدجتس الجديدة.
- تحديث `mockDb.ts` لدعم تخزين الملاحظات، التذكيرات، وتفاعلات العملاء.
- إضافة لوحة جانبية `CalendarEventsPanel` لعرض تفاصيل اليوم عند الضغط على التقويم.
