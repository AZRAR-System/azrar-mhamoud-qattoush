### 1. تقرير الملخص المالي (`financial_summary`)
**قبل الإصلاح:**
```typescript
data: [
  { item: 'إجمالي المحصل', value: 50000 },
  { item: 'إجمالي المتأخر', value: 12000 }
]
```

**بعد الإصلاح:**
```typescript
const installments = get<الكمبيالات_tbl>(KEYS.INSTALLMENTS);
const totalExpected = installments.reduce((sum, inst) => sum + inst.القيمة, 0);
const paidInstallments = installments.filter(i => i.حالة_الكمبيالة === INSTALLMENT_STATUS.PAID);
const totalPaid = paidInstallments.reduce((sum, inst) => sum + inst.القيمة, 0);
```

**البيانات المعروضة:**
- ✅ إجمالي المتوقع (من جميع الأقساط)
- ✅ إجمالي المحصل (الأقساط المدفوعة فعلياً)
- ✅ إجمالي المتأخر (الأقساط المتأخرة)
- ✅ إجمالي القادم (الأقساط المستقبلية)
- ✅ المتبقي (الفرق بين المتوقع والمحصل)
- ✅ نسبة التحصيل (%)
