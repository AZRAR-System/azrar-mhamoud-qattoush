# 🚀 دليل النشر على السيرفر المحلي
## نظام AZRAR - خطوات التشغيل على الشبكة المحلية

---

## 📋 المتطلبات الأساسية

### ما تحتاجه:
- ✅ جهاز كمبيوتر (Windows/Mac/Linux)
- ✅ Node.js مثبت (الإصدار 18 أو أحدث)
- ✅ المشروع الحالي

---

## 🎯 الطريقة الأولى: استخدام Vite (الأسهل والأسرع)

### الخطوة 1: تشغيل النظام
```bash
# افتح Terminal/PowerShell في مجلد المشروع
npm run dev
```

**النتيجة:**
```
VITE v6.4.1  ready in 500 ms

➜  Local:   http://localhost:3000/
➜  Network: http://192.168.1.100:3000/
```

### الخطوة 2: الوصول من الأجهزة الأخرى

**من الكمبيوتر نفسه:**
- افتح المتصفح واكتب: `http://localhost:3000`

**من الموبايل أو التابلت:**
1. تأكد أن الموبايل على نفس شبكة الواي فاي
2. افتح المتصفح واكتب: `http://192.168.1.100:3000`
   (استبدل 192.168.1.100 بالـ IP الذي ظهر لك)

---

## 🏗️ الطريقة الثانية: بناء النظام للإنتاج

### الخطوة 1: بناء المشروع
```bash
npm run build
```

**النتيجة:**
- سيتم إنشاء مجلد `dist` يحتوي على الملفات الجاهزة
- حجم صغير ومحسّن (حوالي 1.2 ميجابايت)

### الخطوة 2: تشغيل السيرفر

**الخيار أ: استخدام serve (بسيط)**
```bash
# تثبيت serve (مرة واحدة فقط)
npm install -g serve

# تشغيل السيرفر
serve -s dist -p 3000
```

**الخيار ب: استخدام http-server**
```bash
# تثبيت http-server (مرة واحدة فقط)
npm install -g http-server

# تشغيل السيرفر
http-server dist -p 3000
```

## 🌐 الطريقة الثالثة: استخدام XAMPP (للمبتدئين)

### الخطوة 1: تثبيت XAMPP
1. حمّل XAMPP من: https://www.apachefriends.org
2. ثبّت البرنامج
3. شغّل Apache من لوحة التحكم

### الخطوة 2: نسخ الملفات
```bash
# بعد عمل npm run build
# انسخ محتويات مجلد dist إلى:
C:\xampp\htdocs\azrar
```

### الخطوة 3: الوصول للنظام
- من الكمبيوتر: `http://localhost/azrar`
- من الموبايل: `http://192.168.1.X/azrar`

---

## 🔍 كيف تعرف IP جهازك؟

### على Windows:
```bash
ipconfig
```
ابحث عن: `IPv4 Address` مثل: `192.168.1.100`

### على Mac/Linux:
```bash
ifconfig
```
ابحث عن: `inet` مثل: `192.168.1.100`

## 📱 الوصول من الموبايل والتابلت

### الخطوات:
1. **تأكد من الاتصال بنفس الشبكة**
   - الكمبيوتر والموبايل على نفس الواي فاي

2. **اعرف IP الكمبيوتر**
   ```bash
   ipconfig  # على Windows
   ```

3. **افتح المتصفح على الموبايل**
   ```
   http://192.168.1.100:3000
   ```

4. **احفظ الصفحة على الشاشة الرئيسية**
   - على iPhone: Share → Add to Home Screen
   - على Android: Menu → Add to Home Screen

**النتيجة:** سيعمل مثل تطبيق مستقل! 📱

## 🛡️ الأمان والحماية

### للشبكة المحلية فقط:
- ✅ النظام آمن داخل شبكتك
- ✅ لا يمكن الوصول من الإنترنت
- ✅ محمي بنظام الصلاحيات

### إذا أردت الوصول من الإنترنت:
⚠️ **لا يُنصح به بدون:**
- استخدام HTTPS
- جدار ناري
- قاعدة بيانات آمنة

## 🔧 حل المشاكل الشائعة

### المشكلة 1: "Port 3000 is already in use"
**الحل:**
```bash
# استخدم منفذ آخر
npm run dev -- --port 3001
```

### المشكلة 2: لا يمكن الوصول من الموبايل
**الحل:**
1. تأكد من نفس الشبكة
2. أوقف جدار الحماية مؤقتاً
3. تأكد من IP الصحيح

### المشكلة 3: الصفحة بيضاء بعد البناء
**الحل:**
```bash
# تأكد من المسار الصحيح في vite.config.ts
base: './'
```
