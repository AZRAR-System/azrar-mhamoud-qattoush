## 💻 أمثلة الكود

### الاستخدام الأساسي
```tsx
const toast = useToast();
toast.success('تمت العملية بنجاح!');
```

### حوار تأكيد
```tsx
const confirmed = await toast.confirm({
  title: 'حذف العنصر',
  message: 'هل أنت متأكد؟',
  isDangerous: true,
  onConfirm: async () => { /* الحذف */ }
});
```

### التحكم في الصوت
```tsx
audioService.setVolume(0.5);
audioService.setEnabled(true);
```
