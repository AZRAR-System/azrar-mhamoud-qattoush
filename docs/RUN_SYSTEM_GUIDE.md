## 🔧 استكشاف الأخطاء

### إذا لم تظهر البيانات:

1. **تحقق من تشغيل الاختبارات:**
  - اذهب إلى `/#/sys-maintenance`
  - اضغط "تشغيل الاختبارات"
  - إذا كنت تريد إضافة بيانات اختبار: فعّل خيار "السماح بإنشاء بيانات اختبار" ثم أعد التشغيل

2. **امسح الـ Cache:**
   - اضغط `Ctrl+Shift+Delete`
   - اختر "حذف كل شيء"
   - أعد تحميل الصفحة

3. **تحقق من Console:**
   - افتح `F12`
   - اذهب إلى `Console`
   - ابحث عن الأخطاء

4. **أعد تشغيل النظام:**
   - أوقف الخادم (`q` في الترمينال)
   - أعد التشغيل (`npm run dev`)
