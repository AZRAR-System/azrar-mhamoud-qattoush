## 🔧 طرق الاستخدام

### **الطريقة 1️⃣: من Admin Panel (موصى به)**

1. اذهب إلى: **الإدارة** → **لوحة التحكم المركزية**
2. انقر على tab **"إدارة النظام"** (أيقونة الترس ⚙️)
3. انقر على الزر الأحمر: **"مسح كل البيانات التجريبية"**
4. أكّد العملية عند الطلب
5. ستُحذف جميع البيانات وتُعاد تحميل الصفحة تلقائياً

**الفائدة:** واجهة آمنة مع تأكيد مزدوج

---

### **الطريقة 2️⃣: من Developer Console (للمطورين)**

افتح **Browser DevTools** (F12) واكتب:

```javascript
// طريقة 1: استدعاء مباشر
resetAllData()

// طريقة 2: عبر DbService
await import('./services/mockDb').then(m => m.DbService.resetAllData())
```

ستظهر رسالة النجاح في Console:
```
✅ تم مسح كامل البيانات التجريبية
📊 البيانات المحفوظة: Users, Roles, Permissions, Lookups, Templates
🗑️  البيانات المحذوفة: 18 جداول
```

**الفائدة:** سريعة وفورية للاختبار

## ⚠️ نقاط مهمة

### 🔴 تنبيهات حرجة:

| التنبيه | الوصف | الحل |
|------|------|------|
| **لا عودة** | لا يوجد Undo بعد المسح | احفظ البيانات قبل المسح |
| **حذف نهائي** | حذف من localStorage وCache | تأكد من التأكيد قبل الضغط |
| **بيانات حساسة** | المستخدمين آمنين ✅ | يمكنك حذف البيانات بأمان |

### ✅ الإجراءات الآمنة:

## 🧪 اختبار Reset

### ✅ خطوات التحقق:

1. **قبل Reset:**
   ```javascript
   // تحقق من البيانات الموجودة
   DbService.getPeople().length  // مثلاً: 10
   DbService.getContracts().length  // مثلاً: 5
   ```

2. **بعد Reset:**
   ```javascript
   // تحقق من الحذف
   DbService.getPeople().length  // يجب = 0
   DbService.getContracts().length  // يجب = 0
   
   // تحقق من الحماية
   DbService.getSystemUsers().length  // يجب > 0 ✅
   ```

3. **النتيجة المتوقعة:**
   ```
   ✅ 0 أشخاص
   ✅ 0 عقارات
   ✅ 0 عقود
   ✅ 0 دفعات
   ✅ 1+ مستخدم (محمي)
   ✅ الصلاحيات سليمة
   ```
