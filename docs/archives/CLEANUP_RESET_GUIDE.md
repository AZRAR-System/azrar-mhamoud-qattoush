## 🎯 متى تستخدم هذه الوظيفة؟

### ✅ **استخدامات مناسبة**:
1. **بعد التطوير/الاختبار**: مسح البيانات التجريبية قبل نشر النظام للعميل
2. **بداية عام مالي جديد**: إعادة تهيئة النظام (بعد أخذ نسخة احتياطية)
3. **تدريب الموظفين**: مسح البيانات التدريبية بعد انتهاء الدورة
4. **تصحيح أخطاء جماعية**: مسح بيانات خاطئة والبدء من جديد

### ❌ **لا تستخدم في**:
1. **بيئة الإنتاج** بدون نسخة احتياطية
2. **بدون موافقة المدير**
3. **لحذف سجل واحد** (استخدم الحذف العادي)
4. **بدون فهم كامل** للنتائج

## 🛠️ طرق التنفيذ

### الطريقة 1️⃣: عبر واجهة المستخدم (موصى بها)

**المسار**: Admin Control Panel → تبويب "إدارة النظام"

**الخطوات**:
1. سجّل دخول كـ **SuperAdmin**
2. افتح صفحة **Admin Control Panel** (`/#/admin`)
3. انتقل لتبويب **"إدارة النظام"**
4. ابحث عن قسم **"مسح البيانات التجريبية"** (أسفل الصفحة)
5. اقرأ التحذير
6. اضغط **"مسح كل البيانات التجريبية"**
7. أكّد في الـ Dialog الأول
8. أكّد مرة أخرى في الـ Dialog الثاني
9. انتظر... (تستغرق ~2 ثانية)
10. الصفحة ستُعاد تلقائيًا

---

### الطريقة 2️⃣: عبر Browser Console (للمطورين)

**الخطوات**:
1. افتح Developer Tools (`F12`)
2. انتقل لتبويب **Console**
3. نفّذ:
```javascript
const result = window.resetAllData();
console.log(result);
```

**النتيجة المتوقعة**:
```javascript
{
  success: true,
  message: "تم مسح جميع البيانات التجريبية بنجاح",
  deletedTables: 18,
  timestamp: "2026-01-15T10:30:00.000Z",
  propertiesReset: true
}
```

## 🐛 استكشاف الأخطاء

### ❌ المشكلة: الزر لا يظهر

**السبب**: المستخدم الحالي ليس SuperAdmin

**الحل**:
1. سجّل خروج
2. سجّل دخول كـ SuperAdmin (username: `admin`, password: `123456`)
3. أعد المحاولة

---

### ❌ المشكلة: "غير مصرح" رغم تسجيل الدخول كـ admin

**السبب**: دور admin مُخزّن خطأ

**الحل**:
```javascript
// تحقق من الدور
const user = JSON.parse(localStorage.getItem('currentUser') || '{}');
console.log('Current role:', user.role);

// إذا كان الدور خطأ، صححه:
user.role = 'SuperAdmin';
localStorage.setItem('currentUser', JSON.stringify(user));
window.location.reload();
```

---

### ❌ المشكلة: البيانات لم تُحذف بالكامل

**السبب**: خطأ في دالة `resetAllData()`

**الحل**:
```javascript
// مسح يدوي
const keys = Object.keys(localStorage).filter(k => k.startsWith('db_'));
keys.forEach(k => {
  if (!k.includes('users') && !k.includes('roles') && !k.includes('lookups') && !k.includes('templates')) {
    localStorage.removeItem(k);
  }
});
window.location.reload();
```
