# 🧪 اختبر Dynamic Engine بنفسك - خطوات التحقق السريعة

## ⚡ اختبار في 5 دقائق

### ✅ الاختبار 1: إنشاء جدول بسيط

```
الخطوات:
1. اذهب إلى Dynamic Builder (#/dynamic-builder)
2. اضغط زر (+) في "الجداول المخصصة"
3. أدخل: "جدول الاختبار"
4. اضغط "إنشاء الجدول"

النتيجة المتوقعة:
✅ الجدول يظهر في القائمة
✅ يمكن اختياره
```

### ✅ الاختبار 2: إضافة حقل للجدول

```
الخطوات:
1. اختر الجدول من القائمة
2. في الأعلى، أدخل:
   - عنوان: "الاسم"
   - برمجي: name
   - النوع: نص
3. اضغط "إضافة"

النتيجة المتوقعة:
✅ الحقل يظهر في القائمة
✅ يصبح جاهزاً للاستخدام
```

### ✅ الاختبار 3: إدخال بيانات

```
الخطوات:
1. في القسم السفلي، أدخل:
   - الاسم: "أحمد محمد"
2. اضغط "حفظ السجل"

النتيجة المتوقعة:
✅ السجل يظهر في الجدول
✅ البيانات محفوظة
```

### ✅ الاختبار 4: التحقق من الحفظ

```
الخطوات:
1. أغلق الصفحة (أو الروابط)
2. عد إلى Dynamic Builder
3. اختر الجدول نفسه

النتيجة المتوقعة:
✅ البيانات لا تزال موجودة
✅ تم حفظها في localStorage
```

## 🔍 الميزات الموجودة (اختبرها):

### ✅ الميزات التي تعمل بشكل مثالي:

```
1. إنشاء جداول جديدة
   ✅ اختبر: أنشئ 3 جداول مختلفة

2. إضافة حقول متعددة
   ✅ اختبر: أضف 5 حقول مختلفة (text, number, date)

3. إدخال سجلات
   ✅ اختبر: أدخل 10 سجلات

4. عرض البيانات
   ✅ اختبر: شاهد الجدول الكامل

5. الحفظ التلقائي
   ✅ اختبر: أعد تحميل الصفحة

6. أنواع الحقول
   ✅ اختبر: استخدم text, number, date

7. واجهة RTL
   ✅ اختبر: كل شيء بالعربية
```

## 🎬 سيناريو اختبار كامل (15 دقيقة)

### السيناريو: بناء جدول "المشاريع"

#### الخطوة 1: الإعداد (2 دقيقة)
```
[ ] اذهب إلى Dynamic Builder
[ ] أنشئ جدول جديد: "المشاريع"
[ ] تحقق: الجدول يظهر في القائمة
```

#### الخطوة 2: إضافة الحقول (5 دقائق)
```
[ ] أضف حقل 1: اسم المشروع (text)
[ ] أضف حقل 2: الميزانية (number)
[ ] أضف حقل 3: تاريخ البدء (date)
[ ] أضف حقل 4: الحالة (text)
[ ] تحقق: 4 حقول ظاهرة
```

#### الخطوة 3: إدخال البيانات (5 دقائق)
```
[ ] سجل 1: "موقع الويب" | 50000 | 2024-01-15 | جاري
[ ] سجل 2: "تطبيق جوال" | 75000 | 2024-02-20 | تخطيط
[ ] سجل 3: "قاعدة بيانات" | 30000 | 2024-03-10 | انتظار
[ ] تحقق: 3 سجلات ظاهرة
```

#### الخطوة 4: التحقق من الحفظ (3 دقائق)
```
[ ] أغلق الصفحة (Ctrl+W أو F5)
[ ] عد إلى Dynamic Builder
[ ] اختر "المشاريع"
[ ] تحقق: البيانات الـ 3 لا تزال موجودة
```

#### النتيجة النهائية:
```
✅ جدول كامل مع 4 حقول
✅ 3 سجلات محفوظة
✅ بيانات دائمة
✅ النظام يعمل ممتاز!
```

## 🐛 الأخطاء المحتملة (وكيفية حلها)

### خطأ 1: الجدول لا يظهر
```
السبب: قد لم تُحفظ البيانات
الحل:
1. تحديث الصفحة
2. افتح DevTools (F12)
3. شاهد localStorage
```

### خطأ 2: البيانات اختفت
```
السبب: قد تم مسح localStorage
الحل:
1. أدخل البيانات من جديد
2. احفظ نسخة احتياطية
3. لا تمسح البيانات يدوياً
```

### خطأ 3: الأزرار لا تعمل
```
السبب: قد يكون حقل فارغ
الحل:
1. تأكد من ملء جميع الحقول
2. تحقق من الأسماء البرمجية
3. لا تستخدم مسافات في الأسماء
```
