```

الصيغة: تذكير ودي وإيجابي
```

### 2. **يوم الاستحقاق** (due)
```

اللون: برتقالي 🟠
الصيغة: تنبيه عاجل اليوم
```

### 3. **تأخير بسيط** (late)
```
الحالة: أقل من أسبوع تأخر
اللون: أحمر 🔴
الصيغة: تنبيه ودي بالدفع
```

### 4. **إنذار رسمي** (warning)
```
الحالة: أسبوع إلى شهر تأخر
اللون: أحمر داكن 🔴🔴
الصيغة: إنذار رسمي قوي
```

### 5. **إشعار قانوني** (legal)
```
الحالة: أكثر من شهر تأخر
اللون: بنفسجي 🟣
الصيغة: إشعار قانوني رسمي
```

## 🔌 التكامل مع صفحة الكمبيالات

### أزرار ذكية في الجدول
في كل سطر كمبيالة، يظهر زر رسالة يختار النوع تلقائياً:

```
حالة الدفعة              الزر الظاهر
───────────────────────────────────────
مسددة بالكامل          ✓ مسددة (معطل)
قبل الاستحقاق > 3 أيام   تذكير (أزرق)
3 أيام قبل              استحقاق (برتقالي)
< 7 أيام تأخر           متأخر (أحمر)
7-30 أيام تأخر          إنذار (أحمر داكن)
> 30 أيام تأخر          قانوني (بنفسجي)
```

### خطوات الاستخدام
1. اذهب إلى `/installments`
2. اختر عقد وانشره
3. انظر لعمود "رسائل" في الجدول
4. اضغط الزر المناسب
5. اختر:
   - 📋 نسخ (Copy)
   - 💬 واتساب (WhatsApp)
   - ✏️ تحرير (Edit)

### مثال كود للاستخدام
```typescript
import { NotificationTemplates } from '@/services/notificationTemplates';

const template = NotificationTemplates.getById('post_late_reminder');
const filled = NotificationTemplates.fillComplete(template, {
  tenantName: 'أحمد محمد',
  amount: 5000,
  daysLate: 7
});

NotificationTemplates.openWhatsApp(
  `${filled.title}\n\n${filled.body}`,
  '966501234567'
);
```
