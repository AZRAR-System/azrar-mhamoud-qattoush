## 🎯 الأساسيات

### **ما هي الكمبيالات؟**
الكمبيالات هي دفعات شهرية أو دورية من المستأجر للمالك.

**الحالات الممكنة:**
- ✅ **مدفوع** - تم السداد بالكامل
- 🔵 **دفعة جزئية** - تم سداد جزء من المبلغ
- ⏳ **غير مدفوع** - لم يتم السداد بعد

### **الإجراءات المتاحة:**
1. **سداد كامل** ✓ - دفع المبلغ كاملاً
2. **دفعة جزئية** ÷ - دفع جزء من المبلغ
3. **عكس سداد** ↩️ - إلغاء السداد السابق

## 📱 الصفحات الرئيسية

### **1️⃣ صفحة الكمبيالات (الدفعات)**
**الرابط:** `http://localhost:3000/installments`

**الميزات:**
- ✅ عرض جميع الكمبيالات مجمعة حسب العقود
- ✅ عرض معلومات المستأجر والعقار
- ✅ عرض تصنيف السلوك المالي
- ✅ الدفع المباشر بدون نقرات إضافية
- ✅ بحث وتصفية متقدمة

**الأزرار الأساسية:**
```
┌─────────────────────────────────┐
│ المستأجر: أحمد علي              │
│ العقار: شقة-2024-A              │
├─────────────────────────────────┤
│ الكمبيالة #1          15000 د.أ  │
│ ✓ سداد كامل | ÷ دفعة جزئية      │
├─────────────────────────────────┤
│ الكمبيالة #2          15000 د.أ  │
│  ✓ سداد كامل | ÷ دفعة جزئية     │
├─────────────────────────────────┤
│ التصنيف: ممتاز (95 نقطة)        │
└─────────────────────────────────┘
```

### **2️⃣ صفحة العمليات (السداد المتقدم)**
**الرابط:** `http://localhost:3000/operations`

**الميزات:**
- ✅ واجهة سداد متقدمة بـ 4 خطوات
- ✅ شرح تفصيلي لكل خطوة
- ✅ معالجة آمنة وتأكيد قبل الحفظ
- ✅ شاشة نجاح مع ملخص العملية

**الخطوات:**
```
الخطوة 1: اختيار العقد
   ↓
الخطوة 2: اختيار الكمبيالة
   ↓
الخطوة 3: إدخال تفاصيل السداد
   ↓
الخطوة 4: التأكيد النهائي
   ↓
✅ تم بنجاح!
```

### **3️⃣ صفحة العمولات**
**الرابط:** `http://localhost:3000/commissions`

**الميزات:**
- ✅ عرض عمولات المالك والمستأجر
- ✅ عرض العمولات الخارجية
- ✅ حسابات تلقائية
- ✅ تقارير شهرية

### **4️⃣ لوحة التحكم الإدارية**
**الرابط:** `http://localhost:3000/admin`

**الميزات:**
- ✅ سجل النشاط (Activity Log) الكامل
- ✅ تتبع جميع العمليات
- ✅ بحث وتصفية
- ✅ معلومات مفصلة عن كل عملية

## 🔧 العمليات المتاحة

### **🟢 السداد الكامل**

**خطوات:**
```
1. اذهب إلى صفحة الكمبيالات
2. اختر العقد المطلوب
3. اضغط على زر "✓ سداد كامل"
4. أكد في نافذة التأكيد
5. انتظر التحديث التلقائي (100ms)
```

**النتائج:**
- ✅ حالة الكمبيالة → "مدفوع"
- ✅ القيمة المتبقية → 0
- ✅ تحديث تصنيف → +5 نقاط
- ✅ إشعار صوتي (صوت النجاح)
- ✅ رسالة توضيحية
- ✅ تسجيل في السجل

### **🔵 الدفعة الجزئية**

**خطوات:**
```
1. اذهب إلى صفحة الكمبيالات
2. اختر العقد المطلوب
3. اضغط على زر "÷ دفعة جزئية"
4. أدخل المبلغ المراد دفعه
5. لاحظ الباقي يُحسب تلقائياً
6. اختر التاريخ (اختياري)
7. أضف ملاحظات (اختياري)
8. اضغط "تأكيد السداد"
```

**النتائج:**
- 🔵 حالة الكمبيالة → "دفعة جزئية"
- 🔵 القيمة المتبقية → المتبقي
- 🔵 تحديث تصنيف → -10 نقاط
- 🔵 إشعار صوتي (صوت التحذير)
- 🔵 رسالة تحذيرية
- 🔵 حفظ الملاحظات
- 🔵 تسجيل في السجل

### **🔴 عكس السداد**

**خطوات:**
```
1. اذهب إلى صفحة الكمبيالات
2. اختر العقد المطلوب
3. ابحث عن كمبيالة مدفوعة (باللون الأخضر)
4. اضغط على زر "عكس السداد"
5. أكد في نافذة التأكيد (بلون أحمر)
6. انتظر التحديث
```

**النتائج:**
- ❌ حالة الكمبيالة → "غير مدفوع"
- ❌ القيمة المتبقية → القيمة الكاملة
- ❌ تحديث تصنيف → -5 نقاط
- ❌ حذف تاريخ الدفع والملاحظات
- ❌ إشعار صوتي (صوت التحذير)
- ❌ رسالة معلومات
- ❌ تسجيل في السجل

## 💡 أمثلة عملية

### **مثال 1: أحمد يدفع الكمبيالة كاملة**

```
السيناريو:
- المستأجر: أحمد علي
- الكمبيالة: #INS-2024-12-001
- المبلغ: 15,000 د.أ
- التاريخ: 2025-01-15

الخطوات:
1. فتح صفحة الكمبيالات
2. البحث عن "أحمد علي"
3. الضغط على "✓ سداد كامل"
4. تأكيد في الحوار

النتيجة:
✅ تم السداد: 15,000 د.أ
✅ التصنيف: ممتاز (95 نقطة)
✅ إشعار: "تم السداد الكامل: 15,000 د.أ"
✅ السجل: مسجل في activity log
```

### **مثال 2: محمد يدفع جزء من الكمبيالة**

```
السيناريو:
- المستأجر: محمد السعودي
- الكمبيالة: #INS-2024-12-002
- المبلغ الأصلي: 12,000 د.أ
- المبلغ المدفوع: 8,000 د.أ
- الملاحظة: "دفع بنكي - والباقي في الشهر القادم"

الخطوات:
1. فتح صفحة الكمبيالات
2. البحث عن "محمد السعودي"
3. الضغط على "÷ دفعة جزئية"
4. إدخال 8000
5. ملاحظة: الباقي = 4,000 (يُعرض تلقائياً)
6. إضافة الملاحظة
7. الضغط على "تأكيد السداد"

النتيجة:
🔵 دفعة جزئية: 8,000 د.أ
🔵 الباقي: 4,000 د.أ
🔵 التصنيف: متوسط (85 نقطة)
🔵 إشعار: "دفعة جزئية: 8,000 د.أ - الباقي: 4,000 د.أ"
🔵 السجل: مسجل مع الملاحظة
```

### **مثال 3: إلغاء سداد خاطئ**

```
السيناريو:
- سداد خاطئ حدث بالأمس
- يجب إلغاء السداد

الخطوات:
1. فتح صفحة الكمبيالات
2. البحث عن الكمبيالة المدفوعة بالخطأ
3. الضغط على "عكس السداد"
4. تأكيد العملية (بلون أحمر)

النتيجة:
❌ تم إلغاء السداد
❌ الحالة: غير مدفوع
❌ التصنيف: انخفض 5 نقاط
❌ السجل: مسجل العملية كاملة
```

## 🔍 استكشاف الأخطاء

### **❓ المشكلة: الزر لا يظهر**

**الحل:**
- ✅ تحقق من حالة الكمبيالة
- ✅ السداد الكامل يظهر فقط للكمبيالات غير المدفوعة
- ✅ عكس السداد يظهر فقط للكمبيالات المدفوعة

### **❓ المشكلة: البيانات لا تتحدث**

**الحل:**
- ✅ صحح الصفحة (F5)
- ✅ تحقق من اتصال الإنترنت
- ✅ افتح console للتحقق من الأخطاء

### **❓ المشكلة: الصوت لا يعمل**

**الحل:**
- ✅ تحقق من مستوى الصوت في النظام
- ✅ تحقق من إعدادات المتصفح
- ✅ جرب متصفح آخر

### **❓ المشكلة: الإشعارات لا تظهر**

**الحل:**
- ✅ تحقق من حقوق الإشعارات
- ✅ صحح الصفحة
- ✅ تحقق من إعدادات المتصفح

## 📊 تقرير البيانات

### **أين توجد البيانات؟**

**1. localStorage:**
```javascript
// الكمبيالات
localStorage['INSTALLMENTS']

// الأشخاص والتصنيفات
localStorage['PEOPLE']

// سجل العمليات
localStorage['OPERATION_LOGS']

// العمولات
localStorage['COMMISSIONS']
```

**2. Admin Panel:**
- اذهب إلى `/admin`
- اختر تبويب "سجل النشاط"
- شاهد جميع العمليات

### **كيفية تصدير البيانات؟**

```javascript
// في DevTools Console:

// تصدير الكمبيالات
JSON.stringify(JSON.parse(localStorage['INSTALLMENTS']), null, 2)

// تصدير السجل
JSON.stringify(JSON.parse(localStorage['OPERATION_LOGS']), null, 2)
```
