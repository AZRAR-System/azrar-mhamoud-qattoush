## 🚀 الاستخدام الفوري

### ابدأ الآن في 3 خطوات:

#### ✅ الخطوة 1: انتقل إلى صفحة Dynamic Builder
```
من القائمة الرئيسية → اختر "Dynamic Builder"
أو اذهب مباشرة إلى: #/dynamic-builder
```

#### ✅ الخطوة 2: أنشئ جدول جديد
```
1. اضغط على زر (+) في "الجداول المخصصة"
2. أدخل اسم الجدول: مثلاً "بيانات العملاء"
3. اضغط "إنشاء الجدول"
```

#### ✅ الخطوة 3: أضف حقول البيانات
```
1. اختر الجدول من القائمة
2. أضف حقول (اسم، بريد، هاتف، تاريخ)
3. ابدأ إدخال البيانات
```
