## 🔗 خريطة الربط الكاملة

### **المستوى 1: الصفحات (UI)**
```
Installments.tsx (صفحة الكمبيالات الرئيسية)
├─ عرض الكمبيالات
├─ ثلاث أزرار: سداد كامل، دفعة جزئية، عكس سداد
├─ معلومات المستأجر والعقار
└─ تصنيف السلوك

Operations.tsx (صفحة السداد المتقدم)
├─ واجهة 4 خطوات
├─ اختيار العقد والكمبيالة
├─ إدخال تفاصيل السداد
└─ تأكيد وإكمال

Commissions.tsx (صفحة العمولات)
├─ عمولات العقود
├─ العمولات الخارجية
└─ حسابات تلقائية

AdminControlPanel.tsx (لوحة التحكم)
├─ تبويب سجل النشاط
└─ عرض جميع العمليات
```

### **المستوى 2: الخدمات (Services)**
```
DbService (mockDb.ts)
├─ getInstallments()          ← قراءة الكمبيالات
├─ getContracts()             ← قراءة العقود
├─ getPeople()                ← قراءة المستأجرين
├─ markInstallmentPaid()      ← تحديث + تسجيل
├─ reversePayment()           ← إلغاء + تسجيل
├─ updateTenantRating()       ← تحديث التصنيف
├─ getLogs()                  ← عرض السجلات
└─ logOperationInternal()     ← تسجيل العملية

notificationService
├─ notify.success()           ← إشعارات النجاح
├─ notify.warning()           ← إشعارات التحذير
├─ toast.success()            ← رسائل النجاح
└─ toast.warning()            ← رسائل التحذير

audioService
└─ playSound()                ← أصوات تنبيهية
```

### **المستوى 3: التخزين (Storage)**
```
localStorage
├─ INSTALLMENTS               ← الكمبيالات
├─ PEOPLE                     ← المستأجرون والتصنيفات
├─ CONTRACTS                  ← العقود
├─ COMMISSIONS                ← العمولات
├─ OPERATION_LOGS             ← سجل العمليات
└─ EXTERNAL_COMMISSIONS       ← العمولات الخارجية
```
