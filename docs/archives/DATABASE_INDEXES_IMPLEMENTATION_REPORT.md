## ✅ التحقق من الصحة

### ✅ منع التكرار حسب نوع العقار:
```typescript
// أرض
checkLandUniqueConstraint('123', '456', 'أرض')
// Result: يمنع تكرار (رقم_القطعة + رقم_اللوحة) للأراضي

// شقة
checkApartmentUniqueConstraint('123', '456', '789', 'شقة')
// Result: يمنع تكرار (رقم_القطعة + رقم_اللوحة + رقم_الشقة) للشقق
```

### ✅ لا يوجد أكثر من عقد ساري لنفس العقار:
```typescript
checkActiveContractUniqueConstraint('PROP-001', 'نشط')
// Result: يمنع إنشاء عقد ساري جديد إذا كان هناك عقد ساري موجود
```

### ✅ الكود الداخلي فريد دائماً:
```typescript
checkPropertyCodeUnique('PROP-001')
// Result: يمنع تكرار الكود الداخلي
```

### ✅ جميع العلاقات تعمل بدون تضارب:
- ✅ الأشخاص ← العقارات (رقم_المالك)
