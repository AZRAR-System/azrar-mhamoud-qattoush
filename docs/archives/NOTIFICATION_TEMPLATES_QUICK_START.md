# نظام نماذج الرسائل والإشعارات - دليل سريع 📧

## ما تم إنجازه ✅

تم إنشاء نظام متكامل وشامل لإدارة نماذج الرسائل والإشعارات مع:

### 1. **الملفات المنشأة:**
- ✅ `src/services/notificationTemplates.ts` - خدمة النماذج الكاملة
- ✅ `src/components/MessageComposer.tsx` - مكون تكوين الرسائل
- ✅ `src/components/panels/NotificationTemplatesPanel.tsx` - لوحة الإدارة
- ✅ `src/pages/NotificationTemplates.tsx` - صفحة الإدارة

### 2. **النماذج الافتراضية:**
1. **تذكير قبل الاستحقاق** - تذكير ودي قبل الموعد
2. **يوم الاستحقاق** - تنبيه في يوم الاستحقاق
3. **تأخير بسيط** - تنبيه بعد التأخر بأيام
4. **إنذار رسمي** - إنذار رسمي بضرورة الدفع
5. **إشعار قانوني** - إشعار بإجراءات قانونية

### 3. **الميزات الرئيسية:**
- 📝 نماذج قابلة للتعديل بالكامل
- {{ }} متغيرات ديناميكية في النماذج
- 💬 تكامل مباشر مع واتساب
- 👁️ معاينة فورية قبل الإرسال
- 💾 حفظ تلقائي في localStorage
- ✅ تفعيل/تعطيل النماذج
- 📊 تصنيف تلقائي للنماذج
- 🔄 إعادة تعيين إلى الافتراضية

### 4. **التكامل مع الواجهة:**
في صفحة الكمبيالات (Installments):
- **أزرار ذكية** تظهر حسب حالة الدفعة:
  - 🔵 تذكير (أزرق) - قبل الاستحقاق
  - 🟠 استحقاق (برتقالي) - يوم الاستحقاق
  - 🔴 متأخر (أحمر) - بعد الاستحقاق
  - 🔴🔴 إنذار (أحمر داكن) - تأخير متوسط
  - 🟣 قانوني (بنفسجي) - تأخير طويل

## البدء السريع 🚀

### 1. **الوصول إلى إدارة النماذج:**
```
http://localhost:3000/notification-templates
```

### 2. **استخدام النماذج من صفحة الكمبيالات:**
```
http://localhost:3000/installments
```

### 3. **خطوات الاستخدام:**
1. اذهب إلى الكمبيالات
2. اختر عقد
3. انظر لعمود "رسائل" 
4. اختر الزر المناسب (تذكير/استحقاق/الخ)
5. اختر الإجراء:
   - 📋 نسخ الرسالة
   - 💬 فتح في واتساب
   - ✏️ تحرير الرسالة

## المتغيرات المدعومة 🔤

استخدم هذه في النماذج:

```
{{tenantName}}        - اسم المستأجر
{{propertyCode}}      - كود العقار
{{amount}}            - المبلغ المستحق
{{dueDate}}           - تاريخ الاستحقاق
{{daysLate}}          - عدد أيام التأخر
{{contractNumber}}    - رقم العقد
{{remainingAmount}}   - المبلغ المتبقي
```

**مثال:**
```
السلام عليكم {{tenantName}}
المبلغ المستحق: {{amount}} د.أ
التاريخ: {{dueDate}}
```

## مثال عملي 💡

### في البرنامج:
```typescript
import { NotificationTemplates } from '@/services/notificationTemplates';

// الحصول على نموذج
const template = NotificationTemplates.getById('post_late_reminder');

// ملء النموذج بالبيانات
const filled = NotificationTemplates.fillComplete(template, {
  tenantName: 'أحمد محمد',
  amount: 5000,
  dueDate: '2025-01-15',
  daysLate: 7,
  propertyCode: 'E-102'
});

// فتح في واتساب
const message = `${filled.title}\n\n${filled.body}`;
NotificationTemplates.openWhatsApp(message, '966501234567');
```

## أهم الإجراءات 🎯

### تحرير نموذج:
1. الذهاب إلى `/notification-templates`
2. اضغط زر التحرير (✏️)
3. عدّل الاسم أو النص
4. احفظ التغييرات

### إضافة نموذج جديد:
1. اضغط "نموذج جديد"
2. أدخل البيانات
3. استخدم `{{ }}` للمتغيرات
4. احفظ

### تعطيل نموذج:
1. اضغط زر العين (👁️)
2. النموذج لن يظهر الآن

### معاينة:
1. اضغط زر المعاينة (👁️)
2. غير البيانات التجريبية
3. شاهد النتيجة فوراً
