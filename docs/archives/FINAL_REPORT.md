## 🔍 نظام التحقق من البيانات

### الدوال المتاحة:

```typescript
// التحقق الشامل من كل شيء
validateAllData(): ValidationResult

// التحقق من المفاتيح الأساسية
checkPrimaryKeyDuplicates(): ValidationResult

// التحقق من القيود الفريدة
checkUniqueConstraints(): ValidationResult

// التحقق من سلامة المفاتيح الخارجية
checkForeignKeyIntegrity(): ValidationResult

// التحقق من قواعد العمل
checkBusinessLogic(): ValidationResult

// التحقق قبل إضافة شخص
validateNewPerson(data): ValidationResult

// التحقق قبل إضافة عقار
validateNewProperty(data): ValidationResult
```

### مثال على الاستخدام:

```typescript
import { validateAllData } from '@/services/dataValidation';

const result = validateAllData();

if (result.isValid) {
  console.log('✅ جميع البيانات صحيحة!');
} else {
  console.error('❌ أخطاء:', result.errors);
  console.warn('⚠️ تحذيرات:', result.warnings);
}
```

## 🗄️ هيكلية قاعدة البيانات

### الجداول الرئيسية:

1. **الأشخاص_tbl** - معلومات الأشخاص (ملاك، مستأجرين، كفلاء)
2. **شخص_دور_tbl** - أدوار الأشخاص (مالك/مستأجر/كفيل/وسيط)
3. **العقارات_tbl** - معلومات العقارات
4. **العقود_tbl** - عقود الإيجار
5. **الكمبيالات_tbl** - الأقساط والدفعات

### العلاقات:

```
الأشخاص_tbl (1) ──────< (N) شخص_دور_tbl
     │
     ├──< (N) العقارات_tbl (رقم_المالك)
     │
     ├──< (N) العقود_tbl (رقم_المستاجر)
     │
     └──< (N) العقود_tbl (رقم_الكفيل)

العقارات_tbl (1) ──────< (N) العقود_tbl (رقم_العقار)

العقود_tbl (1) ──────< (N) الكمبيالات_tbl (رقم_العقد)
```

## ✅ التحققات المُطبّقة

### عند إضافة شخص:
- ✅ الاسم مطلوب ولا يمكن أن يكون فارغاً
- ✅ رقم الهاتف مطلوب ولا يمكن أن يكون فارغاً
- ⚠️ رقم الهاتف يجب أن يبدأ بـ 07 ويتكون من 10 أرقام
- ⚠️ الرقم الوطني يجب أن يتكون من 10 أرقام
- ❌ الرقم الوطني يجب أن يكون فريداً (لا تكرار)

### عند إضافة عقار:
- ✅ الكود الداخلي مطلوب ولا يمكن أن يكون فارغاً
- ✅ المالك مطلوب ويجب أن يكون موجوداً
- ✅ نوع العقار مطلوب
- ✅ العنوان مطلوب
- ❌ الكود الداخلي يجب أن يكون فريداً (لا تكرار)

### عند إنشاء عقد:
- ✅ العقار موجود وشاغر
- ✅ المستأجر موجود في النظام
- ✅ لا يوجد عقد نشط آخر لنفس العقار
- ✅ التواريخ منطقية (النهاية بعد البداية)

### عند حذف سجل:
- ✅ لا يمكن حذف مالك له عقارات
- ✅ لا يمكن حذف شخص له عقود
- ✅ لا يمكن حذف عقار له عقود

## 🚀 كيفية البدء

### 1. تشغيل النظام:

```bash
npm install
npm run dev
```

### 2. الوصول إلى النظام:

```
http://localhost:5173
```

### 3. تسجيل الدخول:

**حساب المدير:**
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

**حساب الموظف:**
- اسم المستخدم: `user`
- كلمة المرور: `user123`

### 4. البدء بإدخال البيانات:

1. **إضافة مالك:**
   - الأشخاص → إضافة شخص → اختر "مالك"

2. **إضافة عقار:**
   - العقارات → إضافة عقار → اختر المالك

3. **إضافة مستأجر:**
   - الأشخاص → إضافة شخص → اختر "مستأجر"

4. **إنشاء عقد:**
   - العقود → إنشاء عقد → اختر العقار والمستأجر
   - ✅ يتم إنشاء الكمبيالات تلقائياً

5. **تسجيل دفعة:**
   - الكمبيالات → اختر الكمبيالة → سجّل الدفعة
