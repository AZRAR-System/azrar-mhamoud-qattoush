# 🚀 دليل استخدام Dynamic Engine - ابدأ الآن!

## ⚡ بدء سريع (2 دقيقة)

### 1. انتقل إلى صفحة Dynamic Builder
```
من القائمة الرئيسية:
اضغط على "Dynamic Builder" أو اذهب إلى #/dynamic-builder
```

### 2. إنشاء جدول جديد
```
1. اضغط على زر (+) في قسم "الجداول المخصصة"
2. أدخل اسم الجدول: مثلاً "العملاء المحتملين"
3. اضغط "إنشاء الجدول"
```

### 3. إضافة حقول للجدول
```
1. اختر الجدول من القائمة
2. اضغط "إضافة حقل جديد"
3. ملئ المعلومات:
   - عنوان الحقل: "اسم العميل"
   - اسم برمجي: customer_name
   - نوع: نص
4. اضغط "إضافة"
```

### 4. إدخال البيانات
```
1. ملئ بيانات الحقول
2. اضغط "حفظ السجل"
3. شاهد البيانات في الجدول أسفل الصفحة
```

## 📋 حالات الاستخدام

### مثال 1: متابعة العملاء الجدد
```
اسم الجدول: العملاء الجدد

الحقول:
- الاسم (نص)
- البريد (نص)
- الهاتف (رقم)
- تاريخ التسجيل (تاريخ)
```

### مثال 2: جدول المهام
```
اسم الجدول: المهام

الحقول:
- عنوان المهمة (نص)
- الوصف (نص)
- تاريخ الاستحقاق (تاريخ)
- الأولوية (نص)
```

### مثال 3: جدول الإجازات
```
اسم الجدول: الإجازات

الحقول:
- اسم الموظف (نص)
- نوع الإجازة (نص)
- من تاريخ (تاريخ)
- إلى تاريخ (تاريخ)
```

## 🎯 أنواع الحقول المتاحة

| النوع | الوصف | مثال |
|------|-------|-------|
| **نص** | نص عادي | اسم، عنوان، ملاحظات |
| **رقم** | أرقام فقط | السعر، العدد، الهاتف |
| **تاريخ** | تاريخ | تاريخ الميلاد، موعد الاجتماع |

## 💡 نصائح مهمة

### ✅ افعل:
```
✅ استخدم أسماء واضحة للحقول
✅ استخدم أسماء برمجية بدون مسافات (customer_name)
✅ اختر النوع الصحيح للحقل
✅ أضف تفاصيل كافية
```

### ❌ تجنب:
```
❌ لا تستخدم أسماء عربية للأسماء البرمجية
❌ لا تترك الحقول فارغة
❌ لا تستخدم أحرف خاصة في الأسماء البرمجية
```

## 🔄 الخطوات بالتفصيل

### إنشاء جدول كامل (خطوة بخطوة)

#### الخطوة 1: إنشاء الجدول
```
1. اذهب إلى Dynamic Builder
2. القسم الأيمن: "الجداول المخصصة"
3. اضغط على أيقونة (+)
4. اكتب: "بيانات المشاريع"
5. اضغط "إنشاء الجدول"
```

#### الخطوة 2: إضافة الحقول (3 حقول)
```
الحقل الأول:
- عنوان: اسم المشروع
- برمجي: project_name
- نوع: نص
- اضغط: إضافة

الحقل الثاني:
- عنوان: الميزانية
- برمجي: budget
- نوع: رقم
- اضغط: إضافة

الحقل الثالث:
- عنوان: تاريخ البدء
- برمجي: start_date
- نوع: تاريخ
- اضغط: إضافة
```

#### الخطوة 3: إدخال سجل
```
ملئ البيانات:
- اسم المشروع: "تطوير الموقع"
- الميزانية: 50000
- تاريخ البدء: 2024-01-15

اضغط: حفظ السجل
```

#### النتيجة:
```
الجدول يعرض صف واحد بالبيانات التي أدخلتها
```

## ❓ الأسئلة الشائعة

### س: أين تُحفظ البيانات؟
```
ج: في localStorage على جهازك
✅ آمن وسريع
⚠️ إذا مسحت البيانات تختفي البيانات
```

### س: هل يمكن حذف جدول؟
```
ج: ليس الآن (في القادم)
🔄 يمكنك إنشاء جدول جديد بدلاً منه
```

### س: هل يمكن تعديل سجل؟
```
ج: ليس الآن (في القادم)
🔄 يمكنك حذف وإضافة جديد
```

### س: كم سجل يمكن إضافة؟
```
ج: حسب حجم localStorage (~5-10MB)
✅ مئات الآلاف من السجلات
```

### س: هل يمكن رجوع البيانات؟
```
ج: لا، لا يوجد نسخة احتياطية تلقائية
💡 احفظ البيانات يدوياً (قريباً)
```

## 🎬 مثال كامل: جدول "فرص البيع"

```
الخطوة 1: إنشاء الجدول
├── اسم: فرص البيع
└── اضغط: إنشاء

الخطوة 2: إضافة الحقول
├── حقل 1: اسم العميل (نص)
├── حقل 2: قيمة الفرصة (رقم)
├── حقل 3: تاريخ المتابعة (تاريخ)
└── حقل 4: الحالة (نص)

الخطوة 3: إدخال بيانات
├── سجل 1: أحمد محمد | 50000 | 2024-02-20 | جديد
├── سجل 2: فاطمة علي | 75000 | 2024-02-22 | متابعة
└── سجل 3: محمود خالد | 100000 | 2024-02-25 | جاهز

النتيجة: جدول يحتوي 3 فرص بيع
```

## 🔍 استكشاف الأخطاء

### المشكلة: الجدول لا يظهر
```
✓ التحقق: هل أنشأت جدول فعلاً؟
✓ الحل: اذهب للصفحة مجدداً
✓ الحل: احذف localStorage وابدأ من جديد
```

### المشكلة: البيانات لا تُحفظ
```
✓ التحقق: هل ملأت الحقول بشكل صحيح؟
✓ التحقق: هل الأنواع صحيحة؟
✓ الحل: حاول مرة أخرى
```

### المشكلة: رسالة خطأ
```
✓ اقرأ الرسالة بعناية
✓ تأكد من الأسماء البرمجية
✓ لا تستخدم مسافات في الأسماء
```
