**جاهز للاستخدام الفوري ✨**

---

## 🎯 الخطوات الأساسية (3 خطوات فقط!)

### 1️⃣ تشغيل الخادم
```bash
npm run dev
# سيفتح http://localhost:3000
```

### 2️⃣ اذهب لصفحة العمليات
```
القائمة الجانبية → العمليات والإجراءات
أو اكتب المباشر: http://localhost:3000/operations
```

### 3️⃣ ابدأ عملية السداد
```
بحث → اختيار العقد → اختيار الدفعة → إدخال البيانات → تأكيد ✓
```

## 📊 ما الجديد؟

### نقاط التحسين الرئيسية:

| الميزة | الحالة | الملاحظة |
|--------|--------|---------|
| 4-خطوات منظمة | ✅ نشط | واجهة سلسة وواضحة |
| تأكيدات صوتية | ✅ نشط | صوت لكل خطوة |
| شريط التقييم | ✅ نشط | أسفل كل عقد في الكمبيالات |
| دفعات جزئية | ✅ نشط | مدعومة مع تنبيهات |
| ملاحظات الدفع | ✅ نشط | اختياري مع التاريخ |

## 🎬 مثال عملي خطوة بخطوة

### السيناريو: سداد دفعة لعقد

```
1. افتح صفحة العمليات
   
2. ابحث عن "أحمد محمد"
   📝 حقل البحث في الأعلى
   
3. اختر عقده من القائمة
   🔊 (ستسمع صوت تأكيد)
   ➡️ تم الانتقال للخطوة 2
   
4. اختر الدفعة من القائمة
   📝 قد ترى "متأخرة 🔴" بجانبها
   🔊 (صوت تأكيد آخر)
   ➡️ تم الانتقال للخطوة 3
   
5. أدخل المبلغ:
   💰 تركه كما هو = دفعة كاملة
   💵 أو أدخل مبلغ أقل = دفعة جزئية
   
6. أدخل التاريخ (الافتراضي = اليوم)
   
7. أضف ملاحظة اختياري
   📝 "دفع بنكي" أو "شيك"
   
8. اضغط "التالي: تأكيد السداد"
   🔊 (صوت التحقق)
   ➡️ تم الانتقال للخطوة 4
   
9. راجع الملخص:
   ✅ اسم المستأجر
   ✅ رقم العقد
   ✅ كود العقار
   ✅ المبلغ المدفوع
   ✅ التاريخ والملاحظات
   
10. اضغط "تأكيد السداد النهائي"
    🔊 (صوت النجاح!)
    
11. شاهد شاشة النجاح:
    🎉 "تم السداد بنجاح!"
    📋 ملخص العملية
    ➡️ سيعود للبداية تلقائياً بعد 3 ثوان
```

## 🎨 المظهر والألوان

### حالات الخطوات:

```
🔵 الخطوة الحالية   → أزرق مع تفاصيل
✅ خطوة مكتملة     → أخضر مع علامة اختيار
⚪ خطوة لم تصل     → رمادي خافت
```

### شريط التقييم:

```
🟢 ممتاز (90-100)   → أخضر متدرج + ✓
🔵 جيد (60-89)     → أزرق متدرج + ✓
🟠 متوسط (30-59)   → برتقالي مع ⚠️
🔴 ضعيف (10-29)    → أحمر مع ❌
🔴 سيء (0-9)       → أحمر داكن مع 🚫
```

| الحالة | الصوت | متى |
|--------|-------|-----|
| اختيار عقد | ✓ مرتفع | عند الضغط على العقد |

| خطأ في الإدخال | ✗ عالي | مبلغ غير صحيح |
| تحقق من البيانات | ✓ متوسط | عند الانتقال |
| نجاح السداد | 🎉 احتفالي | عند تأكيد السداد |

## ⚠️ الحالات الخاصة

### 1. دفعة متأخرة
```
📌 سترى علامة حمراء "متأخرة" بجانب الدفعة
⏰ الدفعات القديمة من التاريخ الحالي تُعتبر متأخرة
🔔 ستتأثر بسلبية درجة المستأجر
```

### 2. دفعة جزئية
```
💵 أدخل مبلغاً أقل من الحد الأقصى
⚠️ ستشاهد تحذير: "دفعة جزئية - الباقي: XXX د.أ"
🔴 ستنخفض درجة المستأجر (-10 نقاط)
📝 الباقي سيظهر في القيمة_المتبقية
```

### 3. ملاحظات مهمة
```
📝 يمكنك إضافة ملاحظات مثل:
   - "دفع شيك"
   - "تحويل بنكي"
   - "دفع نقدي"
   - "دفعة مستأجرة أخرى"
💾 ستُحفظ في قاعدة البيانات
```

## 🔍 الفحص والتحقق

### تحقق من تحديث البيانات:

```
1. بعد السداد مباشرة:
   📊 اذهب لصفحة "الكمبيالات"
   👀 الدفعة يجب أن تظهر "مدفوع" ✓
   
2. شريط التقييم:
   ⭐ يجب أن يتحدث حسب السداد:
      - دفعة كاملة: +5 نقاط
      - دفعة جزئية: -10 نقاط
      - دفعة متأخرة: -20 نقاط
   
3. في localStorage:
   F12 → Application → LocalStorage
   ابحث عن "khbni_db"
```

## 🐛 معالجة المشاكل الشائعة

### ❌ لا تسمع الأصوات؟
```
✅ تحقق من صوت المتصفح (ليس كتم الصوت)
✅ تحقق من الكونسول: F12 → Console
✅ أعد تحميل الصفحة: F5
✅ امسح الذاكرة المؤقتة: Ctrl+Shift+Delete
```

### ❌ البيانات لم تتحدث؟
```
✅ تحقق من وحدة التحكم (Console)
✅ تأكد من انتقاء العقد والدفعة بشكل صحيح
✅ جرب تحديث الصفحة بـ F5
✅ تحقق من localStorage في DevTools
```

### ❌ الخطوات لا تتقدم؟
```
✅ اقرأ رسائل الخطأ في الأعلى
✅ تأكد من إدخال مبلغ صحيح (> 0)
✅ لا تحاول إدخال مبلغ أكبر من الحد الأقصى
✅ جرب إعادة تشغيل الخادم: npm run dev
```
