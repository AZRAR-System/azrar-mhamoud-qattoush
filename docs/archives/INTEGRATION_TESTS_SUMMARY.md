## 🚀 التشغيل

### من الواجهة الرسومية
```
1. اذهب إلى /integration-tests
2. اضغط "تشغيل الاختبارات"
3. راقب النتائج
4. حمّل التقرير
```

### من DevTools
```javascript
// في متصفح Chrome/Firefox
await runTests();

// أو
const suite = new IntegrationTestSuite();
const results = await suite.runAllTests();
console.log(results);
```

### من الكود
```typescript
import { IntegrationTestSuite } from './services/integrationTests';

async function main() {
  const suite = new IntegrationTestSuite();
  const results = await suite.runAllTests();
  
  // معالجة النتائج
  results.forEach(result => {
    console.log(`${result.testName}: ${result.status}`);
  });
}

main();
```
