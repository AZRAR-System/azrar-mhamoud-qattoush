## 🚀 خطة النشر على السيرفر المحلي

### الخطوة 1: بناء النظام
```bash
npm run build
```
**النتيجة:** مجلد `dist` يحتوي على الملفات الجاهزة

### الخطوة 2: تثبيت سيرفر محلي
**الخيار الأول (سهل):**
```bash
npm install -g serve
serve -s dist -p 3000
```

**الخيار الثاني (احترافي):**
```
تثبيت XAMPP أو WAMP
نسخ مجلد dist إلى htdocs
فتح المتصفح على localhost
```

### الخطوة 3: الوصول من الأجهزة الأخرى
```
1. معرفة IP الجهاز: ipconfig (Windows)
2. فتح المتصفح على الموبايل/التابلت
3. كتابة: http://192.168.1.X:3000
```
