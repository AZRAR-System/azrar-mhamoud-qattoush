## 🚀 الخطوات التالية

### المطلوب من المطورين:

#### 1. تطبيق الملفات الجديدة
```bash
# 1. استبدال .gitignore
mv .gitignore.new .gitignore

# 2. استبدال README
mv README-NEW.md README.md

# 3. إضافة التبعيات الجديدة
npm install --save-dev jest @testing-library/react @testing-library/jest-dom ts-jest
npm install --save electron-log

# 4. تحديث package.json
# أضف السكريبتات:
"scripts": {
  "test": "jest",
  "test:watch": "jest --watch",
  "test:coverage": "jest --coverage",
  "lint": "eslint src electron",
  "logs:view": "type %APPDATA%\\azrar-desktop\\logs\\azrar-*.log"
}
```

#### 2. إعداد GitHub Secrets
في GitHub Repository Settings > Secrets:
```
CSC_LINK=<base64-encoded-certificate>
CSC_KEY_PASSWORD=<certificate-password>
SNYK_TOKEN=<snyk-api-token>
DISCORD_WEBHOOK=<discord-webhook-url>
```

#### 3. تحديث روابط GitHub
استبدل `your-username` في:
- `electron-builder.config.js`
- `README-NEW.md`
- `.github/workflows/build.yml`
- `CHANGELOG.md`

#### 4. الحصول على شهادة توقيع
- اطلب شهادة Code Signing من DigiCert/Sectigo
- اتبع دليل `docs/CODE_SIGNING.md`
